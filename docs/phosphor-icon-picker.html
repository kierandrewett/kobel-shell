kobel-shell — Phosphor icon picker

This records the original custom-glyph to Phosphor migration map. Active control glyphs are vendored from phosphor-icons/core v2.0.8 Bold under crates/kobel-theme/assets/icons/ and render at 16 logical pixels. Use the variant grid to compare alternatives; selections autosave to this browser (localStorage) but do not rewrite the Rust assets. Application icons in the dock/launcher/tray come from the system icon theme and are not affected.
