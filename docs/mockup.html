kobel-shell — the full chrome suite, mocked

Design direction for the Quickshell chrome that owns everything gnoblin vacates: bar, dock, control centre, notifications, OSD, launcher, session. Goblin-green on green-cast ink, pill geometry, springy motion.

The shell is live too — click the status cluster for the control centre, the clock for notifications, the bell to ring the OSD. Everything dismisses on scrim click, exactly like the real thing will.
